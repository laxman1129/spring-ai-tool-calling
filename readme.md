## Spring AI function calling

## prerequisites

- Ollama installed
- Api ninjas API key from [here](https://api-ninjas.com/)

## How to run

- using intellij idea or command line
    ```shell
    mvn spring-boot:run
    ```

## use
```shell
GET http://localhost:8080/api/weather?question=what is the weather in Bangalore
```
